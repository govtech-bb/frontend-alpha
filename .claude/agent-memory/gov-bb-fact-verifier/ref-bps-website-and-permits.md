## Barbados Police Service (BPS) — confirmed 2026-05-28

**Correct website:** https://tbps.gov.bb (The Barbados Police Service) — actively maintained, content from 2025–2026.
**Wrong domain:** https://police.gov.bb — ECONNREFUSED; do not use.

**HQ address:** Former Barclays Bank Complex, Lower Roebuck Street, Bridgetown, St. Michael (confirmed on gov.bb and tbps.gov.bb)

**Phone:** (246) 430-7100 (PBX); Commissioner Secretary: 430-7106; Crime Stoppers: 429-8787

**BPS-administered permits (per official BPS Facebook post):**
- Loud music permits
- Occasional liquor licences
- Both require applications submitted **4 weeks** before event date

**"Public gathering licence":** This name for a BPS permit is NOT confirmed in any public source — not on tbps.gov.bb, not in any Act, not in any gov.bb page. The BPS Facebook post on permits does not mention it. Any content using this term needs agency confirmation before publishing.
